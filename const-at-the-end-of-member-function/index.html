const 放在 member function 的後面

const 放前面與後面的意義

  • 放前面：代表回傳值是 const 的。
  • 放後面：僅出現在成員函數，代表函數不會改變呼叫者的值；目的是確保 const 的物件不會因為呼叫了某個 function 而變化。

光文字敘述可能不太清楚，下面來舉栗子。
